IceVim

A stability-focused Neovim fork

What is IceVim?

IceVim is a Neovim fork frozen at version 0.12. No new features. No API changes. Only security patches and critical bug fixes.

Your config from today will work for years.

Installation

git clone https://github.com/icevim-editor/icevim.git
cd icevim
./build.sh
